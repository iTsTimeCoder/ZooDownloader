## ✨ Features

* 🎥 Download video & audio from 1000+ websites
* 🎧 Supports MP3, FLAC, WAV, AAC, OGG
* 📺 High quality downloads (up to 4K)
* 📝 Subtitle support (multi-language)
* 📦 Batch downloading
* ⚡ Fast and lightweight
* 🖥️ Simple and clean GUI

---

## ⚙️ Requirements

This project depends on:

* yt-dlp
* FFmpeg

Make sure they are installed and added to PATH.

---

## 🚀 Installation

1. Download the latest release
2. Extract the ZIP file
3. Run `ZooDownloader.exe`

---

## 🧠 Usage

1. Paste video URL
2. Choose format & quality
3. Click Download
